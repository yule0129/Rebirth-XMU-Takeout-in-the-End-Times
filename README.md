# Rebirth-XMU-Takeout-in-the-End-Times
以厦门大学为地图的像素风小游戏
# 中文名：重生之我在末日厦大送外卖

## 项目简介

《重生之我在末日厦大送外卖》是一款使用 Python 和 Pygame 制作的 2D 俯视角像素风小游戏。

游戏背景设定在末日来临后的厦门大学翔安校区。同学们被困在宿舍和教学区域中，玩家作为外卖员，需要在丧尸、极端天气和复杂校园道路中穿梭，把外卖从取餐点送到指定地点，尽可能拯救更多同学。

## 游戏目标

玩家需要在地图中找到当前刷新的取餐点，领取外卖后按照提示前往送达点。

每成功完成一单可获得分数；如果订单超时会扣分。玩家有三条生命，被丧尸抓到会受伤并回到出生点，生命耗尽后游戏结束。

## 操作方式

- `W A S D`：控制角色移动
- `E` 或 `空格`：取餐 / 送达外卖 / 交互
- `空格` 或 `R`：游戏结束后重新开始
- `ESC`：退出游戏

## 主要玩法

1. 根据屏幕提示前往取餐点。
2. 靠近取餐点后按 `E` 或 `空格` 接取订单。
3. 按照黄色目标提示前往送达点。
4. 注意躲避丧尸追逐。
5. 在规定时间内完成订单获得分数。
6. 通过地图连接点切换不同校区场景。

## 游戏特色

- 厦门大学翔安校区主题地图
- 多张地图场景切换
- 像素风校园建筑与道路
- 随机刷新取餐点
- 跨地图和同地图配送任务
- 丧尸巡逻与追逐机制
- 极端天气系统
- 三条生命与受伤 / 倒地动画
- 游戏结束后可重新开始
- 顶部 HUD 显示时间、分数、生命、天气和任务状态

## 地图内容

游戏目前包含多张校园地图，包括但不限于：

- 一期
- 思源
- 丰庭药学区
- 德旺芙蓉区

地图中包含教学楼、宿舍园区、餐厅、湖泊、操场、篮球场、草地、道路和传送点等区域。不同地形会影响通行，例如湖泊不可通过，草地会降低移动速度。

## 运行方法

请先确保电脑已经安装 Python，并安装依赖：

```bash
pip install -r requirements.txt
```

然后运行：

```bash
python main.py
```

如果使用项目自带虚拟环境，可以在项目目录中运行：

```bash
venv\Scripts\python.exe main.py
```

## 计分规则

- 成功送达一单：加 10 分
- 订单超时失败：扣 5 分
- 生命耗尽：游戏结束
