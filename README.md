# 重生之我在末日厦大送外卖

这是一个 Python + pygame 写的 2D 俯视角像素风小游戏。玩家在末日厦大校园中接外卖、送往宿舍，并躲避丧尸和极端天气。

## 运行方法

1. 安装 pygame：

```bash
pip install pygame
```

2. 进入本文件夹，运行：

```bash
python main.py
```

## 操作方式

- `WASD` 或方向键：连续移动
- `E` 或空格：接外卖 / 送达外卖
- `ESC`：退出游戏
- 游戏结束后按 `R`：重新开始

## 游戏提示

- 橙色闪烁光圈是校门取餐点。
- 黄色闪烁光圈是当前订单的送达点。
- 屏幕上会显示虚线引导方向。
- 顶部进度条会显示当前订单剩余时间。
- 被丧尸撞到会损失生命，并丢掉当前外卖。

## 当前已经实现

- 厦门大学主要区域的方块地图：校门、芙蓉湖、宿舍、食堂、操场、道路和建筑
- 更顺滑的连续移动手感
- 清晰的取餐点、送达点和路线引导
- 随机宿舍订单与订单倒计时
- 成功送达加分，超时扣分
- 丧尸巡逻与追逐
- 暴雨、台风、浓雾三种随机天气
- 生命值、得分、已救人数统计
- 游戏结束与重新开始界面
